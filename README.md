# 🌍 World Countries Data Scraping & Analytics

An automated data pipeline that extracts global country demographics from the web and turns raw data into a clean, interactive dashboard.

## 🛠️ Tools Used
* **Python** (BeautifulSoup, Requests, Pandas) - For hunting and gathering the data.
* **Excel & Power Query** - For cleaning the numbers and building the charts.

## 🔄 How I Built It
1. **Scraped the Web:** Wrote a Python script to look inside the website's code (`div.country`) and pull out the data automatically.
2. **Cleaned the Data:** Loaded the CSV into Excel using Power Query, verified data types, and calculated population metrics.
3. **Designed the Dashboard:** Built PivotTables and charts to instantly show the world's largest countries by both land area and population.

## 💡 Cool Insights Found
* **Land Giants:** Russia and Antarctica take up the biggest physical space on our planet.
* **People Giants:** China and India have massive populations compared to the rest of the world, easily taking the top spots on the charts.
